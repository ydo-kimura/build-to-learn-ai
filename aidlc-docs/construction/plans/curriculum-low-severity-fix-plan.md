# カリキュラム重要度「低」修正計画 (Low-Severity Fix Plan)

**作成日** : 2026-07-02
**背景** : 4観点レビューで検出された重要度「低」の指摘のうち、curriculum-quality-fix-plan.md（高・中対応）で未対応だった約30件を修正する。

**修正方針** :

- 既存の「例え話 → 図解 → 実装 → 実践 → 答え合わせ」フォーマットと文体（です・ます調）を維持
- 太字は `**テキスト**`（前後半角スペース）、図の直前に「下図は〜」ブリッジ文の規約を遵守
- 「入門向けの意図的な簡略化」は削除せず、 **注記の追加** で正確性を担保する（教材を難しくしない）
- alt テキストは「図解：<日本語の内容説明>」形式に統一
- `en/curriculum/` と `aidlc-docs/`（本計画ファイル・audit.md 除く）は編集しない
